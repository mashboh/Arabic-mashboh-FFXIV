مهم جداً لصديقك:

بعد فك الضغط يجب أن ترى هذا الشكل بالضبط:

Echoglossian\
  Echoglossian.dll
  Echoglossian.json
  Font\
    NotoSansArabic-Medium.ttf   <--- ضروري جداً (بدون هذا يظهر ====)
    ...

انسخ المجلد كاملاً إلى:
%AppData%\XIVLauncher\devPlugins\Echoglossian

خطأ شائع: نسخ الملفات فقط بدون مجلد Font
أو عمل مجلد مزدوج: devPlugins\Echoglossian\Echoglossian\

ثم فعّل في XIVLauncher:
Settings -> Dalamud -> Load custom plugins from the DevPlugins folder
