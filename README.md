# Arabic - mashboh (FFXIV)

ترجمة واجهة وحوار Final Fantasy XIV إلى العربية + خط عربي اختياري عبر Penumbra.

## التثبيت / التحديث

1. `/xlsettings` → Experimental → Custom Plugin Repositories → أضف الرابط ثم اضغط + ثم Save:

```
https://raw.githubusercontent.com/mashboh/Arabic-mashboh-FFXIV/main/dalamud-repo.json
```

بديل إذا احتجت CDN:

```
https://cdn.jsdelivr.net/gh/mashboh/Arabic-mashboh-FFXIV@main/dalamud-repo.json
```

2. Plugin Installer → **Arabic - mashboh** → Install أو **Update**
3. ثبّت Penumbra وفعّل Enable Mods
4. من إعدادات المود فعّل «تفعيل خط العربية (Penumbra)»

الإصدار الحالي: **4.2601.723.240**
Release: https://github.com/mashboh/Arabic-mashboh-FFXIV/releases/tag/v4.2601.723.240
